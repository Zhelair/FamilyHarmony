# Family Harmony

Working name for the family app rewrite.

Why this name:
- warm
- home-centered
- a little magical
- not childish
- fits progress, rituals, and shared family life

## Direction

Family Harmony is a family dashboard with a soft game layer.

It should feel like:
- a real household app first
- beautiful from day one
- warm and premium
- playful in details, not chaotic

## Core Pillars

- real family accounts
- personal challenges
- shared household goals
- house progression
- fast daily check-ins
- strong mobile UX

## Main References

- Cozi style practicality
- FamilyWall style family dashboard
- light Habitica-style motivation
- sobriety-tracker style streaks and milestones where needed

## Initial Build Choice

- React
- Vite
- Tailwind CSS
- Supabase
- Vercel

## Notes

- `Table & Tide` remains the prototype/reference world
- `Family Harmony` is the v2 rewrite workspace
